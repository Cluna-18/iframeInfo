What are iframes?


Iframes are used to embed a web page within a web page.

Syntax

Here is how an iframe is set up. <iframe src="url" title="description"></iframe>

How it looks on a page

Setting Height and Width

Height and width can be set in the iframe tag. The default is 300px by 150px.

Here is an example of how to change the height and width:

<iframe src="url" title="description" width="500" height="315"></iframe>

Here is an example of changing the height and width to better fit the page.

Adding a border

We can add a border around the frame and adjust the size, style, and color of the iframe's border.

Here is an example of what the code may look like:

<iframe src="url" title="description" width="800" height="1000" style="border-radius: 5%; border:3px solid red;" ></iframe>

Target for a link

We can also use iframes as a target for a link, such that when users click a link, it opens up within the iframe. We can do this by adding a name attirbute to the iframe and making sure a corresponding href link sets the iframe as a target. This is how it may look.

<iframe src="url" title="description" name="iframe_name" title = "Iframe Example" ></iframe>

<a href="url" target="iframe_name">Click here</a>

Click here to open up the quiz in the target iframe below

Note that the iframe is not a link, but a target for the link. The link opens up in the iframe. It also keeps the same style attributes for the different sources.
