Youtube in iframes

One interesting thing we can do with iframes are embedding Youtube videos into them. You can see an example of this below.

One thing to note is that embedding a YouTube video is not as simple as copying and pasting the link to the video into the "src" attribute of the iframe. Rather, YouTube has special code that is generated for each video that allows it to be embedded in an iframe. As such, there are some steps you need to follow in order to obtain this embedding link. These are outlined below.

Getting code to embed a YouTube video:

  1. Go to the Youtube video you want to embed
  2. Click on the share button
  3. Click on the embed button
  4. Copy the code that is generated
  5. Paste it into your HTML file
  6. Change the width and height to fit your page or any style attributes as normal

    7. Here is the what the code may look like:

    <iframe width="560" height="315" src="https://www.youtube.com/embed/kcfs1-ryKWE?si=Nu83ApFQAW5lvF7W" title="Youtube video player"allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
